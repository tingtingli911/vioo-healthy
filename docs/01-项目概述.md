# 项目概述

## 项目名称
**VIOO 健康管理系统** (VIOO Healthy Management System)

## 项目定位
VIOO 是一个个人健康管理系统，旨在帮助用户记录、管理和追踪个人健康数据，提供健康分析和建议。

## 项目目标
1. **数据管理**：提供便捷的健康数据录入和管理功能
2. **数据分析**：对健康数据进行统计分析和可视化展示
3. **健康提醒**：提供健康提醒和建议功能
4. **用户友好**：提供简洁美观的用户界面和良好的用户体验

## 核心特性
- ✅ 用户注册和登录认证
- ✅ Token 认证机制（7天有效期）
- ✅ 超级管理员权限系统（首个注册用户自动成为超级管理员）
- ✅ 用户信息管理
- 🔄 健康数据管理（开发中）
- 🔄 数据统计分析（开发中）
- 🔄 健康提醒功能（开发中）

## 技术特点
- **前后端分离**：Vue 3 前端 + Django REST Framework 后端
- **RESTful API**：标准化的 API 接口设计
- **Token 认证**：安全的用户认证机制
- **响应式设计**：适配不同设备屏幕
- **容器化部署**：支持 Docker 部署

## 项目状态
- **当前版本**：v1.0.0
- **开发阶段**：基础功能已完成，核心功能开发中
- **维护状态**：活跃开发中

## 用户角色
1. **普通用户**：可以管理自己的健康数据
2. **超级管理员**：拥有平台管理权限，可以查看平台统计信息
